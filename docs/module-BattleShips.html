(inner) findNewlySunkShip(state, newFiredCells, cellId) → {Array.<string>|undefined}

Description:

  • Finds the cell array of a ship newly sunk by the latest hit, or undefined. A ship is newly sunk when its first cell is not already in sunkenShips, it contains the hit cell, and R.all of its cells are now in firedCells. Uses R.find so no index tracking is required.
